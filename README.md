# micro1-converter

a simplecurrency converter.

> Favorite currency pairs.

> Convert value.


## Built with Next, Tailwind and Shadcn/ui

## Install

```sh
cd .\converter\
npm install
```

## Usage

```sh
cd .\converter\
npm install
npm run dev
```

Spins up local servers on port 3000


Example:

```sh
Local:     http://localhost:3000
```
